## Project setup
Após baixar ou clonar o projeto, entre na pasta do mesmo e instale o node usando o npm install.
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
